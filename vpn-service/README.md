# VPN Service - Профессиональная платформа для продажи VPN

## 🏗 Архитектура

```
vpn-service/
├── bot/              # Telegram бот (Aiogram 3.x)
├── api/              # REST API для веб-интерфейса и приложений
├── web/              # Сайт выдачи MTProto прокси (5 минут)
├── worker/           # Фоновые задачи (очистка, проверки)
├── shared/           # Общие модули (шифрование, БД, утилиты)
├── ansible/          # Playbook для развертывания
├── docker/           # Docker Compose конфигурация
├── configs/          # Шаблоны конфигураций Xray
└── scripts/          # Скрипты обслуживания
```

## 🔐 Система безопасности

1. **Шифрование подписок** - AES-256 для конфигураций
2. **Привязка к устройству** - по fingerprint клиента
3. **Скрытие данных** - пользователи видят только кнопку подключения
4. **Временные MTProto** - 5 минут жизни, автоудаление
5. **Анти-сплош** - защита от передачи аккаунтов

## 🚀 Особенности

- **Мгновенная выдача** - подписки создаются за <1 сек
- **Автопродление** - напоминания за 24 часа до конца
- **Реферальная система** - 10% с каждого приглашенного
- **Статистика** - трафик, скорость, устройства
- **Мультиязычность** - RU/EN/UK

## 📦 Технологии

- **Backend**: Python 3.12, FastAPI, Aiogram 3.x
- **БД**: MariaDB + Redis (кэш, сессии)
- **Прокси**: Xray-core, MTProto сервер
- **Фронт**: HTML+JS (минималистичный)
- **Инфра**: Docker, Ansible, Caddy
